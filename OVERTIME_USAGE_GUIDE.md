# 📖 دليل استخدام صفحة ساعات العمل الإضافية وأيام الجمعة

## 🎯 نظرة عامة

صفحة **overtime.html** توفر نظاماً متكاملاً لإدارة:
- ✅ **الساعات الإضافية** — تسجيل وإدارة ساعات العمل الإضافية
- ✅ **أيام الجمعة** — طلبات العمل يوم الجمعة والمدفوعات

---

## 🔐 المتطلبات الأساسية

### 1. المصادقة (Authentication)
- يجب تسجيل الدخول أولاً
- سيتم فحص الجلسة تلقائياً عند فتح الصفحة
- إذا لم تكن لديك جلسة، ستُحول تلقائياً لصفحة تسجيل الدخول

### 2. الأدوار المدعومة
| الدور | الصلاحيات |
|------|-----------|
| **surveyor** (مساح) | إضافة ساعات، طلب جمعة، استلام الحساب |
| **head** (رئيس مساحين) | الموافقة على الطلبات، تسجيل الدفع، تصدير الكشوفات |
| **admin** (مدير عام) | الموافقة النهائية على الساعات المعتمدة من الرئيس |

---

## 🕐 دليل الساعات الإضافية

### 👨‍💼 للمساح (Surveyor)

#### الخطوة 1: اختيار الشهر والسنة
```
اختر الشهر → اختر السنة → سيتم تحميل قائمة العمال
```

#### الخطوة 2: إدخال الساعات الإضافية
```
لكل عامل:
- اسم العامل: (مدرج بالقائمة)
- رقم الهوية: (تلقائي)
- الساعات: (أدخل الرقم، مثال: 10.5 ساعة)
```

#### الخطوة 3: حفظ أو رفع
```
✓ حفظ كمسودة — احفظ التعديلات دون إرسال
✓ رفع للرئيس — ارسل السجل لموافقة الرئيس
```

#### الملاحظات
- يمكنك تعديل المسودات قبل الرفع
- بعد الرفع، ستحصل على إشعار بالتقدم
- السجلات المعتمدة لا يمكن تعديلها

### 👔 لرئيس المساحين (Head)

#### عرض الطلبات المعلقة
```
الفلاتر المتاحة:
- الشهر (اختياري)
- السنة (اختياري)  
- الحالة (بانتظار، معتمد، مرفوض)
```

#### الموافقة أو الرفض
```
لكل طلب:
[ ✓ اعتماد ]  — وافق على الطلب
[ ✗ رفض ]     — ارفض مع إمكانية كتابة ملاحظة
[ 👁️ تفاصيل ] — عرض تفاصيل العمال والساعات
```

#### تصدير الملفات
```
بعد الموافقة من المدير:
[ 📊 Excel ] — تصدير ملف تقرير شامل
```

### 👨‍💻 للمدير (Admin)

#### الموافقة النهائية
```
يرى جميع السجلات المعتمدة من الرئيس
ويمكنه الموافقة النهائية
```

## 🎉 دليل أيام الجمعة

### 👨‍💼 للمساح (Surveyor)

#### القسم 1: حسابات أيام الجمعة (استلام الحساب)

**ملخص سريع:**
```
إجمالي الجمع  : X يوم
مستلمة      : Y يوم ✓
غير مستلمة   : Z يوم ⏳
```

**للعمل:**
```
1. اختر الشهر والسنة (اختياري)
2. اختر العامل من القائمة
3. عرض أيام الجمعة المعتمدة
4. عند استلام المبلغ كاش:
   [ 💰 استلمت الحساب ] — تسجيل الاستلام مع ختم رسمي
```

#### القسم 2: طلب يوم جمعة جديد

**الخطوة 1: فتح نموذج الطلب**
```
[ + طلب جمعة جديد ] — فتح نموذج إضافة طلب جديد
```

**الخطوة 2: اختيار التاريخ والعمال**
```
- تاريخ الجمعة: (يجب أن يكون يوم جمعة فعلاً)
- العمال المشاركون: 
  ☑️ العامل 1
  ☑️ العامل 2
  إلخ...
```

**الخطوة 3: إضافة عمال مستعارين (اختياري)**
```
1. أدخل الرقم الوظيفي للمساح الآخر
2. ابحث عن المساح
3. اختر العامل من قائمة عماله
4. [ + إضافة ]
```

**الخطوة 4: الإرسال**
```
[ 📤 إرسال الطلب ] — إرسال الطلب لرئيس المساحين
```

### 👔 لرئيس المساحين (Head)

#### القسم 1: الطلبات الواردة
```
الفلاتر:
- الشهر
- السنة
- الحالة (بانتظار، موافق، مرفوض)

لكل طلب:
[ ✓ موافقة ] [ ✗ رفض ] [ 👁️ تفاصيل ]
```

#### القسم 2: متابعة المدفوعات
```
جدول يبين:
- المساح
- الرقم الوظيفي
- إجمالي الأيام
- المدفوعة
- المتبقية

يمكنك تسجيل الدفع:
[ 💰 تسجيل دفع ] لكل طلب معتمد
```

#### القسم 3: تصدير الكشف
```
[ 📊 تصدير كشف المدفوعات ]

ينتج ملف Excel يحتوي على:
- ملخص لكل مساح
- تفاصيل كل يوم جمعة
- حالة الدفع
- بيانات الختم
```


## 🐛 استكشاف الأخطاء

### المشكلة: لا أرى قائمة العمال
**الحل:**
1. تأكد من أن المساح (أنت) لديك عمال مسجلون
2. اذهب لصفحة "العمال" وأضف عمالاً
3. ارجع لصفحة الساعات وأعد تحميل الصفحة

### المشكلة: طلبي لا يظهر للرئيس
**الحل:**
1. تأكد من الضغط على "رفع للرئيس" وليس فقط "حفظ"
2. أعد تحميل الصفحة
3. تحقق من الإشعارات أو تواصل مع الرئيس

### المشكلة: لا أستطيع تحديث سجل معتمد
**الحل:**
هذا صحيح — السجلات المعتمدة مقفولة
إذا كان هناك خطأ، اطلب من المدير فتح السجل مجدداً

### المشكلة: خطأ عند التصدير
**الحل:**
1. تأكد من تثبيت مكتبة XLSX (تحميل تلقائي)
2. جرب متصفح آخر (Chrome أفضل)
3. تأكد من عدم استخدام غير متوافق
