- `employee-ids.html` - إدارة الأرقام الوظيفية ⭐ جديد
- `users.html` - إدارة الموظفين
- `devices.html` - إدارة الأجهزة
- `custody.html` - إدارة العهدة
- `branches.html` - إدارة الفروع
- `maintenance.html` - إدارة الصيانة
- `calibration-alerts.html` - تنبيهات المعايرة
- `device-history.html` - سجل تنقل الأجهزة
- `user-custody-history.html` - سجل عهدة الموظف
- `notification-center.html` - مركز الإشعارات
- `logs.html` - سجل العمليات
- `settings.html` - الإعدادات

### ملفات JavaScript
- `app-core.js` - الوظائف الأساسية (تم تحديثه ⭐)
- `main.js` - وظائف إضافية
- `example.js` - أمثلة

### ملفات CSS
